자주 묻는 질문

무료 AI 서비스에 대해 사람들이 가장 많이 묻는 질문들을 모았습니다.

ChatGPT는 무료로 사용할 수 있나요? +
네, ChatGPT는 무료 플랜을 제공합니다. GPT-5 모델을 일일 제한적으로 사용할 수 있으며, 한도 소진 시 GPT-5 mini 모델로 전환되어 무제한 사용 가능합니다. 이미지 업로드 및 분석, 웹 검색, 코드 실행 기능도 무료로 제공되므로, 간단한 작업부터 고급 분석까지 폭넓게 활용할 수 있습니다.
무료 AI 챗봇 중 가장 추천하는 것은? +
목적에 따라 다릅니다. 범용 사용에는 ChatGPT, 장문 문서 분석에는 Claude, 구글 생태계 활용에는 Gemini, 한국어 최적화에는 뤼튼(Wrtn), 논리 추론에는 DeepSeek이 각각 강점을 보입니다. 한 가지에 의존하기보다 여러 서비스를 용도에 따라 함께 사용하는 것이 가장 효과적입니다. 일주일 내내 한도 알림 없이 사용하려면 DeepSeek과 뤼튼이 가장 안정적입니다.
뤼튼(Wrtn)은 무료로 사용할 수 있나요? +
네, 뤼튼(Wrtn)은 한국 AI 플랫폼으로 GPT 및 Claude 계열 모델과 이미지 생성 기능을 매우 넉넉한 무료 한도로 제공합니다. 한국어에 최적화되어 있어 한국 사용자에게 가장 추천되는 무료 AI 서비스 중 하나이며, 별도의 결제 정보 없이 가입만 하면 바로 사용할 수 있습니다.
무료 AI 이미지 생성 사이트를 추천해주세요 +
Canva AI는 무료 플랜에서 AI 이미지 생성, 배경 제거, 이미지 확장 등 다양한 기능을 제공합니다. Ideogram은 텍스트가 깔끔하게 들어간 이미지 생성에 특화되어 하루 5회 무료 생성이 가능합니다. ChatGPT 무료 버전에서도 DALL-E 3 기반 이미지 생성을 제한적으로 사용할 수 있습니다. Microsoft Copilot 무료 버전에서는 DALL-E 3로 일 15장까지 이미지를 생성할 수 있어 가장 넉넉한 편입니다.
AI 서비스 무료 플랜의 한도는 어떻게 되나요? +
각 서비스마다 무료 플랜의 한도가 다릅니다. ChatGPT는 GPT-5 일일 메시지 제한 후 mini 모델로 전환, Claude는 일일 메시지 제한, DeepSeek은 매우 넉넉한 무료 한도, Perplexity는 기본 검색 무제한 및 Pro 검색 일 5회, GitHub Copilot은 월 2,000회 자동완성과 50회 채팅, ElevenLabs는 월 30분 무료 음성 합성을 각각 제공합니다.
Perplexity AI는 무료로 사용할 수 있나요? +
네, Perplexity AI는 무료 플랜에서 기본 검색을 무제한으로 사용할 수 있습니다. 각 답변마다 5~10개의 출처 링크를 함께 제공해 정보의 신뢰도를 확인할 수 있어, 보고서나 논문 자료 수집에 매우 유용합니다. Pro 검색(고급 AI 모델 기반)은 일 5회로 제한됩니다.
GitHub Copilot 무료로 쓸 수 있나요? +
네, GitHub Copilot은 개인 사용자에게 무료 티어를 제공합니다. 월 2,000회의 코드 자동완성과 50회의 AI 채팅을 무료로 사용할 수 있어, VS Code 등 주요 IDE에서 코딩 생산성을 크게 향상시킬 수 있습니다. 인증된 GitHub 계정만 있으면 바로 시작할 수 있습니다.
Groq AI는 무엇이 특별한가요? +
Groq의 가장 큰 특징은 속도입니다. LPU(Language Processing Unit)라는 자체 개발 칩으로 AI 추론을 실행하여, 다른 서비스보다 10배 이상 빠른 응답 속도를 자랑합니다. 무료 티어에서도 Llama 3.3 70B 등 최신 모델을 30 RPM, 일 14,400회까지 사용할 수 있으며, 신용카드 등록이 필요 없습니다.
Cursor와 Windsurf의 차이는 무엇인가요? +
둘 다 AI 기반 코드 에디터이지만 접근 방식이 다릅니다. Cursor는 VS Code 기반에 Claude/GPT 모델을 탑재하여 채팅형 코딩에 강점이 있고, Windsurf는 Cascade라는 에이전트가 자동으로 코드를 분석하고 수정합니다. 두 서비스 모두 무료 플랜을 제공하므로, 직접 사용해보고 자신의 작업 스타일에 맞는 것을 선택하는 것이 좋습니다.
NotebookLM은 어떤 도구이고 무료인가요? +
NotebookLM은 Google의 AI 리서치 어시스턴트로 완전 무료입니다. PDF, 웹페이지, YouTube 영상 등 최대 50개 출처를 업로드하면 AI가 해당 내용만 기반으로 답변하여 환각(hallucination)이 없습니다. Audio Overviews 기능으로 연구 자료를 팟캐스트로 변환해 들을 수 있어 논문, 리포트, 계약서 분석에 혁신적입니다. 구글 계정만 있으면 즉시 사용 가능합니다.
Poe AI로 모든 AI 모델을 무료로 쓸 수 있나요? +
Poe는 GPT-5.5, Claude 4.5, Gemini 3.1, Grok, DeepSeek 등 모든 주요 AI 모델을 한 곳에서 사용할 수 있는 플랫폼입니다. 무료 플랜에서 일 3,000포인트(약 150회 메시지)를 제공합니다. 각 서비스에 개별 가입할 필요 없이 하나의 계정으로 모든 모델을 비교하며 사용할 수 있어, 어떤 AI가 특정 작업에 가장 적합한지 테스트하기에 최적입니다.
Hugging Face Spaces는 무엇인가요? +
Hugging Face Spaces는 50만 개 이상의 오픈소스 AI 모델을 무료로 체험하는 플랫폼입니다. Stable Diffusion으로 이미지 생성, Llama로 챗봇, Whisper로 음성 인식 등 모든 종류의 AI를 브라우저에서 바로 테스트할 수 있습니다. 회원가입 없이도 대부분 사용 가능하며, 최신 AI 연구 결과를 가장 빠르게 직접 체험해볼 수 있는 곳입니다.
